Adding Assets

  • Manual multi-file uploading
  • Setting up a Dropbox Hotfolder
  • HotFolders – You can setup a DropBox Hot Folder in EnterMedia by going to Settings/ Hot Folders (under Asset Settings). Click Add Hot Folder then use this as the External Path: /opt/entermediadb/clients/home/${content.clientdomainprefix}/Dropbox/(Folder Name)
    • Note #1: (Folder Name) is a subfolder within DropBox that contains assets you want to be automatically imported.
    • Note #2: You will also want to make sure that the Importing is scheduled to run on a regular basis. (Recommended setting: 10m)
  • Libraries/ Projects – Setup a library/ project using a Hot Folder path of  (Folder Name)
